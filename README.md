# Analizador de Encabezados de Solicitud

Segundo proyecto del curso **API y Microservicios** de [freeCodeCamp](https://www.freecodecamp.org/).

## Descripción
Este microservicio devuelve información sobre el cliente que realiza la solicitud HTTP, como su dirección IP, idioma preferido y software.

## Uso
**Endpoint principal:**  
`/api/whoami`

**Ejemplo de respuesta JSON:**
```json
{
  "ipaddress": "181.25.44.12",
  "language": "es-AR",
  "software": "Mozilla/5.0 (Windows NT 10.0; Win64; x64)..."
}
